🔥 Brasa Burguer — Site Institucional + E-commerce
Site completo de hamburgueria artesanal com cardápio interativo, carrinho de compras e sistema de pedidos via WhatsApp.

📋 Sobre o Projeto
Site desenvolvido para a Brasa Burguer, uma hamburgueria artesanal fictícia de Porto Alegre. O projeto simula um ambiente real de e-commerce para restaurantes, com foco em experiência do usuário, design moderno e funcionalidade completa de pedidos.

✨ Funcionalidades

🍔 Cardápio interativo com 4 categorias: Burgers, Combos, Acompanhamentos e Bebidas
🛒 Carrinho de compras com controle de quantidade e remoção de itens
🛵 Opção de Entrega com cálculo de frete e busca automática de endereço por CEP
🏪 Opção de Retirada no balcão com tempo estimado
💳 Formas de pagamento: PIX, Crédito, Débito e Dinheiro (com campo de troco)
📱 Finalização via WhatsApp com resumo completo do pedido
🔍 Modal de detalhes por item com controle de quantidade
📍 Busca de CEP automática via API ViaCEP
📲 Layout responsivo para mobile e desktop


🗂️ Estrutura de Arquivos
brasa-burguer/
├── index.html   # Estrutura e marcação HTML
├── style.css    # Estilos, tema, animações e responsividade
└── script.js    # Lógica do cardápio, carrinho e pedidos

🚀 Como Usar

Faça o download ou clone o repositório
Coloque os 3 arquivos na mesma pasta
Abra o index.html no navegador

Não é necessário instalar dependências ou rodar servidor — é HTML puro.

🎨 Design

Tema: Dark com paleta fogo — laranja, vermelho e dourado sobre fundo escuro
Fontes: Bebas Neue (display) + Barlow (corpo)
Animações: Hambúrguer flutuante, fumaça na grelha, marquee, cards com entrada suave
Responsivo: Mobile-first, adaptado para qualquer tela
